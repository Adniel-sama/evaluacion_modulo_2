Consejos para protegerte en línea

  • Usa contraseñas únicas y un gestor de contraseñas.
  • Activa autenticación de dos factores (2FA) siempre que sea posible.
  • Mantén el software y el sistema operativo actualizados.
  • Realiza copias de seguridad periódicas y verifica que puedas restaurarlas.
  • Evita conectar dispositivos USB desconocidos y descarga solo de fuentes confiables.

Test de seguridad

Responde 5 preguntas rápidas para evaluar prácticas básicas.

Test de seguridad
  1. ¿Usas contraseñas únicas para cada servicio?
  2. ¿Tienes activada la autenticación de dos factores (2FA) en cuentas críticas?
  3. ¿Realizas copias de seguridad periódicas de tus archivos importantes?
  4. ¿Abres adjuntos o ejecutas archivos de remitentes desconocidos?
  5. ¿Mantienes tus dispositivos y aplicaciones actualizadas?
