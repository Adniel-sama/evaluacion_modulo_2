# Sitio informativo de Ciberseguridad

## Estructura
- `index.html` — Página de inicio con introducción.
- `informacion.html` — Explicación de amenazas (phishing, ransomware, malware, DDoS) y casos reales en acordeón.
- `consejos.html` — Buenas prácticas, formulario de contacto (validación) y modal con test de seguridad.
- `styles.css` — Estilos mínimos complementarios a Bootstrap.
- `script/script.js` — Validaciones, test y pequeñas interacciones (jQuery + Bootstrap).
- `.gitignore` — Ignora archivos temporales.
- `README.md` — Documentación.

## Tecnologías
- HTML
- CSS + Bootstrap 5
- JavaScript
- Git GitHub
